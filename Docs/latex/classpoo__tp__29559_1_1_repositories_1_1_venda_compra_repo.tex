\doxysubsection{Descrição detalhada}
Repositório especializado na manipulação de vendas e compras. 

A classe {\ttfamily Venda\+Compra\+Repo} é uma especialização da classe base {\ttfamily Base\+Repo$<$Venda\+Compra$>$}, responsável pela persistência e manipulação de objetos {\ttfamily Venda\+Compra}. A classe lida com operações específicas para vendas e compras, armazenando os dados em um ficheiro JSON.
