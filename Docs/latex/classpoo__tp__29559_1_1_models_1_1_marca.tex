\doxysubsection{Descrição detalhada}
Representa uma marca no sistema. 

A classe {\ttfamily Marca} armazena informações sobre uma marca, como seu nome, descrição, data de criação, país de origem, e a sua relação com os produtos. Ela oferece funcionalidades para verificar se a marca pode ser eliminada, com base na sua associação com produtos vendidos.
