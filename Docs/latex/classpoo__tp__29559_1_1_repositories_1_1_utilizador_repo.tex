\doxysubsection{Descrição detalhada}
Repositório para a manipulação de utilizadores. 

A classe {\ttfamily Utilizador\+Repo} herda de {\ttfamily \doxyref{Base\+Repo}{p. }{class_base_repo}$<$\doxyref{Utilizador}{p. }{class_utilizador}$>$} e é especializada na manipulação de objetos do tipo {\ttfamily \doxyref{Utilizador}{p. }{class_utilizador}}. Ela oferece funcionalidades adicionais para procurar utilizadores pelo NIF ou Contacto, bem como obter todos os utilizadores clientes a partir de um ficheiro JSON.
