\doxysubsection{Descrição detalhada}
Repositório especializado na manipulação de utilizadores. 

A classe {\ttfamily Utilizador\+Repo} é uma especialização da classe base {\ttfamily Base\+Repo$<$Utilizador$>$}, responsável pela persistência e manipulação de utilizadores. A classe lida com operações específicas para objetos do tipo {\ttfamily Utilizador}, incluindo a busca por NIF, contacto e a obtenção de todos os utilizadores clientes.
