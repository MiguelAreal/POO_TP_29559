\doxysubsection{Descrição detalhada}
Formulário para exibição de detalhes de uma transação de venda ou compra. 

A classe {\ttfamily Detalhes\+Venda\+Compra} representa uma interface gráfica que exibe os detalhes de uma transação, incluindo informações sobre o cliente, data da transação, valores, garantia, itens comprados e campanhas aplicadas. Este formulário é implementado utilizando o Metro\+Framework.
