\doxysubsection{Descrição detalhada}
Formulário para adicionar ou atualizar um cliente existente. 

Este formulário permite ao utilizador adicionar um novo cliente ou atualizar um cliente existente, fornecendo informações como nome, contacto, morada, NIF, data de nascimento e outros. Utiliza o {\ttfamily \doxyref{Utilizador\+Controller}{p. }{class_utilizador_controller}} para manipular os dados do cliente, garantindo que não existam duplicados e que os dados sejam válidos.
