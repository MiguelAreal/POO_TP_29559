\doxysubsection{Descrição detalhada}
Representa um produto no sistema. 

A classe {\ttfamily \doxyref{Produto}{p. }{class_produto}} armazena informações detalhadas sobre um produto, incluindo o nome, a categoria, a marca, o preço, a quantidade em stock e a data de adição. Esta classe é usada para representar os produtos no sistema e permite a gestão das informações associadas a eles.
