\doxysubsection{Descrição detalhada}
Repositório genérico para manipulação de dados. 

A classe {\ttfamily Base\+Repo$<$T$>$} implementa um repositório genérico para realizar operações de CRUD (Criar, Ler, Atualizar, Apagar) em itens do tipo {\ttfamily T}. Os itens são carregados e guardados em ficheiros JSON, garantindo persistência entre as execuções da aplicação.
