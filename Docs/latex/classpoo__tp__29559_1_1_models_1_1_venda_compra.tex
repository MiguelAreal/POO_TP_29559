\doxysubsection{Descrição detalhada}
Representa uma transação de venda ou compra realizada. 

A classe {\ttfamily Venda\+Compra} armazena informações sobre a transação de compra ou venda de produtos, incluindo o cliente associado, os itens vendidos, o valor total, as datas relevantes, o método de pagamento utilizado, entre outras informações. Esta classe pode ser usada para representar tanto vendas de produtos quanto compras realizadas por um cliente. 

Definido na linha \textbf{ 17} do ficheiro \textbf{ Venda\+Compra.\+cs}.
