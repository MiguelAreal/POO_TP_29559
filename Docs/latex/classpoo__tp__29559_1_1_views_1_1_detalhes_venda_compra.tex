\doxysubsection{Descrição detalhada}
Formulário para exibição de detalhes de uma transação de venda ou compra. 

Esta classe é responsável por exibir informações detalhadas de uma transação de venda ou compra, incluindo dados do cliente, valores totais, itens comprados, campanhas aplicadas e status de garantia.
