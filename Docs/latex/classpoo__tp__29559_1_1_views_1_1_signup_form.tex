\doxysubsection{Descrição detalhada}
View responsável pelo formulário de registo. 

A View {\ttfamily Signup\+Form} permite ao utilizador criar uma nova conta, validando todos os campos necessários, como NIF, contacto e data de nascimento. Possui também integração com o controlador de utilizadores para verificar se os dados fornecidos são únicos.
