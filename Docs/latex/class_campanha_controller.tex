\doxysubsection{Descrição detalhada}
Controlador para a gestão de campanhas. 

A classe {\ttfamily \doxyref{Campanha\+Controller}{p. }{class_campanha_controller}} implementa a lógica para a gestão de campanhas, incluindo operações de leitura e manipulação de dados. Herda de {\ttfamily \doxyref{Base\+Controller}{p. }{class_base_controller}$<$Campanha$>$} e implementa a interface {\ttfamily \doxyref{IEntity\+Controller}{p. }{interface_i_entity_controller}}, oferecendo funcionalidades específicas para o trabalho com campanhas, incluindo a tradução de identificadores para nomes legíveis na interface de utilizador.
