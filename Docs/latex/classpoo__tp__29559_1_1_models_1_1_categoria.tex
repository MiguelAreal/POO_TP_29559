\doxysubsection{Descrição detalhada}
Representa uma categoria de produtos. 

A classe {\ttfamily Categoria} é utilizada para armazenar informações relacionadas a uma categoria de produtos no sistema. Cada categoria possui um identificador único, nome, descrição e data de criação. Além disso, a classe oferece um método para verificar se a categoria pode ser eliminada, o que é possível apenas caso não esteja associada a nenhum produto.

Verifica se a categoria pode ser eliminada. 

Este método verifica se a categoria pode ser eliminada, garantindo que a categoria não esteja associada a nenhum produto. Caso a categoria esteja associada a algum produto, ela não pode ser eliminada.
