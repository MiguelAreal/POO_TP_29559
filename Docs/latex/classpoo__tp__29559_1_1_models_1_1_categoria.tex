\doxysubsection{Descrição detalhada}
Representa uma categoria de produtos no sistema. 

A classe {\ttfamily Categoria} armazena informações como o nome, descrição e data de criação de uma categoria. Também oferece funcionalidade para verificar se a categoria pode ser eliminada, garantindo que não esteja associada a nenhum produto.
