Classe utilitária para manipulação de enums. 

A classe {\ttfamily Enum\+Helper} fornece métodos úteis para trabalhar com tipos enumerados (enums), especialmente para recuperar as descrições associadas a cada valor de um enum. Este utilitário é útil para obter as descrições de enums com o atributo {\ttfamily Description}.
