Classe base para o repositório genérico com operações CRUD. 

A classe {\ttfamily \doxyref{Base\+Repo}{p. }{class_base_repo}$<$T$>$} implementa um repositório genérico para a manipulação de dados com operações de criação, leitura, atualização e remoção (CRUD). Os itens são armazenados em formato JSON num ficheiro, permitindo a persistência dos dados entre as execuções da aplicação.
