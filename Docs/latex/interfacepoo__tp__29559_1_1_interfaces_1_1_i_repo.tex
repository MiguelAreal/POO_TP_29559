\doxysubsection{Descrição detalhada}
Interface para a definição de operações de um repositório genérico. 

A interface {\ttfamily IRepo$<$T$>$} define os métodos essenciais para a manipulação de dados de um repositório genérico, onde {\ttfamily T} representa o tipo de entidade armazenada. A interface pode ser implementada por repositórios que armazenam dados em diferentes fontes, como base de dados ou ficheiros.
