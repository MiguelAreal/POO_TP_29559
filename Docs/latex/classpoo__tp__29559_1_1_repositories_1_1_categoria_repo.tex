\doxysubsection{Descrição detalhada}
Repositório para a manipulação de categorias. 

A classe {\ttfamily Categoria\+Repo} herda de {\ttfamily \doxyref{Base\+Repo}{p. }{class_base_repo}$<$Categoria$>$} e é especializada na manipulação de objetos do tipo {\ttfamily Categoria}. Ela oferece as funcionalidades básicas de um repositório, como a adição, remoção, atualização e obtenção de categorias a partir de um ficheiro JSON.
