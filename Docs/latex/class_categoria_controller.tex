\doxysubsection{Descrição detalhada}
Controlador para a gestão de categorias. 

A classe {\ttfamily \doxyref{Categoria\+Controller}{p. }{class_categoria_controller}} implementa a lógica para a gestão das categorias, incluindo operações de leitura, remoção e verificação de condições para eliminação de categorias associadas a produtos. Herda de {\ttfamily \doxyref{Base\+Controller}{p. }{class_base_controller}$<$Categoria$>$} e implementa a interface {\ttfamily \doxyref{IEntity\+Controller}{p. }{interface_i_entity_controller}}, oferecendo funcionalidades específicas para o trabalho com categorias.
