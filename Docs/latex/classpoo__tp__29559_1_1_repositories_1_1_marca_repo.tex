\doxysubsection{Descrição detalhada}
Repositório para a manipulação de marcas. 

A classe {\ttfamily Marca\+Repo} herda de {\ttfamily Base\+Repo$<$Marca$>$} e é especializada na manipulação de objetos do tipo {\ttfamily Marca}. Ela oferece as funcionalidades básicas de um repositório, como a adição, remoção, atualização e obtenção de marcas a partir de um ficheiro JSON.
