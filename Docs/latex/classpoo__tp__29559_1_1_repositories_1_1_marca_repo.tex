\doxysubsection{Descrição detalhada}
Repositório para a manipulação de marcas. 

A classe {\ttfamily Marca\+Repo} herda de {\ttfamily \doxyref{Base\+Repo}{p. }{class_base_repo}$<$\doxyref{Marca}{p. }{class_marca}$>$} e é especializada na manipulação de objetos do tipo {\ttfamily \doxyref{Marca}{p. }{class_marca}}. Ela oferece as funcionalidades básicas de um repositório, como a adição, remoção, atualização e obtenção de marcas a partir de um ficheiro JSON.
