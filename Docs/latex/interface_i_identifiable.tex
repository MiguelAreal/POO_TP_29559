Interface para garantir que a classe possui um campo {\ttfamily Id}. 

A interface {\ttfamily \doxyref{IIdentifiable}{p. }{interface_i_identifiable}} exige que as classes que a implementem possuam um campo {\ttfamily Id}, que é utilizado para identificar de forma única cada item dentro do sistema.
