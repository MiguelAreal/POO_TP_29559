\doxysubsection{Descrição detalhada}
Interface para a definição de operações de repositório genérico. 

Esta interface define os métodos básicos que devem ser implementados para a gestão de dados de uma entidade do tipo genérico {\ttfamily T}. As operações incluem a obtenção, adição, remoção e atualização de itens no repositório.
