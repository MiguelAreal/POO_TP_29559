\doxysubsection{Descrição detalhada}
Interface para a definição de operações de controle de entidades genéricas. 

Esta interface define os métodos básicos para a gestão de entidades, como filtrar, obter, atualizar e remover itens. ~\newline
 É genérica e pode ser usada com diferentes tipos de entidades na aplicação.
