\doxysubsection{Descrição detalhada}
Enumeração para os tipos de formulários disponíveis na aplicação. 

Esta enumeração define os tipos de formulários que a aplicação utiliza para a gestão de diferentes entidades, como produtos, categorias, marcas, entre outros. A enumeração é utilizada em várias partes da aplicação para identificar e aceder aos diferentes tipos de formulários.
