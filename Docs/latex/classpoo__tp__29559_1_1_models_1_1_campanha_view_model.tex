\doxysubsection{Descrição detalhada}
Representa a visão de uma campanha para o utilizador. 

A classe {\ttfamily Campanha\+View\+Model} é utilizada para exibir ao utilizador as informações relativas a uma campanha, traduzindo as propriedades da classe {\ttfamily Campanha} para uma forma mais acessível. Além disso, esta classe inclui a tradução do identificador da categoria para o nome da categoria, proporcionando uma experiência de utilizador mais amigável.
