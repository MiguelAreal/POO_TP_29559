Representa a visão de uma campanha para o utilizador. 

A classe {\ttfamily Campanha\+View\+Model} traduz as informações da entidade {\ttfamily Campanha} para uma forma mais acessível e amigável, incluindo o nome da categoria em vez do seu identificador.
