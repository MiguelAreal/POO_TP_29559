\doxysubsection{Descrição detalhada}
Representa um utilizador no sistema. 

A classe {\ttfamily \doxyref{Utilizador}{p. }{class_utilizador}} armazena informações pessoais de um cliente ou administrador, incluindo o nome, contacto, morada, NIF, data de nascimento, e outras propriedades associadas ao utilizador. Esta classe é usada para gerir as informações de um utilizador no sistema.
