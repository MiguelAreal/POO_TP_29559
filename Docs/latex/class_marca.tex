\doxysubsection{Descrição detalhada}
Representa uma marca. 

A classe {\ttfamily \doxyref{Marca}{p. }{class_marca}} armazena as informações de uma marca, incluindo o nome, descrição, data de criação, país de origem e a sua relação com produtos. Esta classe permite gerir as marcas associadas a produtos no sistema.

A classe oferece funcionalidades para verificar se a marca pode ser eliminada, com base na sua associação com produtos vendidos, e permite a definição de uma data de criação automática.
