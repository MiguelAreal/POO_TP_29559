\doxysubsection{Descrição detalhada}
Formulário para adicionar um novo produto. 

Este formulário permite ao utilizador adicionar um novo produto ao sistema. Ele solicita informações sobre o nome do produto, a categoria, a marca, o preço e a quantidade em stock. Utiliza o {\ttfamily \doxyref{Produto\+Controller}{p. }{class_produto_controller}} para adicionar o produto ao sistema, e os controladores {\ttfamily \doxyref{Marca\+Controller}{p. }{class_marca_controller}} e {\ttfamily \doxyref{Categoria\+Controller}{p. }{class_categoria_controller}} para carregar as marcas e categorias disponíveis.
