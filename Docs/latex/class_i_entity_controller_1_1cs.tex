\doxysubsection{Descrição detalhada}
Interface para a definição de operações de controlo de entidades genéricas. 

Esta interface define os métodos básicos para a gestão de entidades, permitindo a filtragem, obtenção, atualização e remoção de itens. A interface é genérica, permitindo ser utilizada com diferentes tipos de entidades na aplicação.
