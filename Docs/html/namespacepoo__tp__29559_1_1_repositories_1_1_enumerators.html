enum poo_tp_29559.Repositories.Enumerators.FormTypes

Enumeração que representa os tipos de formulários na aplicação.

A enumeração FormTypes contém os diferentes tipos de formulários utilizados na aplicação para a gestão de várias entidades. Cada valor da enumeração corresponde a um tipo específico de formulário, sendo utilizado para realizar operações de manipulação e visualização de dados em diferentes áreas da aplicação.

Valores de enumerações
Produtos 

Formulário para gestão de produtos.

Este tipo de formulário é utilizado para realizar operações de criação, leitura, atualização e remoção (CRUD) de produtos na aplicação.

Categorias 

Formulário para gestão de categorias de produtos.

Este tipo de formulário permite realizar operações de CRUD sobre as categorias de produtos.

Marcas 

Formulário para gestão de marcas de produtos.

Utilizado para realizar operações de CRUD sobre as marcas associadas aos produtos.

Vendas 

Formulário para gestão de vendas.

Este tipo de formulário é utilizado para a gestão de transações de vendas, permitindo o controlo e registo das vendas realizadas.

Utilizadores 

Formulário para gestão de utilizadores da aplicação.

Permite a gestão de utilizadores do sistema, incluindo operações como criação, atualização e remoção de utilizadores.

Campanhas 

Formulário para gestão de campanhas promocionais.

Utilizado para gerir as campanhas promocionais dentro da aplicação, incluindo a criação, monitorização e edição de campanhas.

Compras 

Formulário para gestão de compras.

Este tipo de formulário é utilizado para gerir as compras realizadas, permitindo a adição, edição e remoção de registos de compras na aplicação.

Enumeração que representa os métodos de pagamento na aplicação.

A enumeração MetodoPagamento define os métodos de pagamento disponíveis para as transações na aplicação, incluindo pagamentos através de débito, crédito e numerário.

Valores de enumerações
Debito 

Método de pagamento por débito.

Representa o pagamento através de cartão de débito.

Credito 

Método de pagamento por crédito.

Representa o pagamento através de cartão de crédito.

Numerario 

Método de pagamento por numerário.

Representa o pagamento realizado em dinheiro (numerário).
