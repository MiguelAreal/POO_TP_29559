Descrição detalhada

Repositório genérico para manipulação de dados.

A classe BaseRepo<T> implementa um repositório genérico para realizar operações de CRUD (Criar, Ler, Atualizar, Apagar) em itens do tipo T. Os itens são carregados e guardados em ficheiros JSON, o que garante persistência entre as execuções da aplicação.

Parâmetros de template
TTipo genérico representando a entidade armazenada no repositório, que deve implementar a interface IIdentifiable.
Restrições do Tipo
T :class 
T :IIdentifiable
