Descrição detalhada

Formulário para a criação ou atualização de uma campanha.

Este formulário serve para criar uma nova campanha ou atualizar uma existente no sistema. O utilizador preenche o nome, a percentagem de desconto, as datas de início e fim, e a categoria associada.
