Descrição detalhada

Representa um produto no sistema.

A classe Produto armazena informações detalhadas sobre um produto, como nome, categoria, marca, preço, quantidade em estoque e data de adição. É usada para representar produtos dentro do sistema e permite a gestão das informações associadas a eles.
