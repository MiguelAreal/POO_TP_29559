Descrição detalhada

Interface para a definição de operações de um repositório genérico.

Parâmetros de template
TTipo genérico que representa a entidade armazenada no repositório. Deve ser uma classe.

A interface IRepo<T> especifica os métodos essenciais para a manipulação de dados em um repositório genérico. Pode ser implementada por repositórios que interagem com diversas fontes de dados, como bases de dados ou ficheiros.
