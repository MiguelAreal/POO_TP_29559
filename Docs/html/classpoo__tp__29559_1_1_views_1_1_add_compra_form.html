Descrição detalhada

Formulário para a criação de uma compra, por um cliente.

Este formulário serve para adicionar uma nova compra ao sistema. O utilizador cliente deve preencher vários campos, como produtos, quantidades e método de pagamento. Além disso, o sistema valida os campos para garantir que os dados inseridos sejam válidos.
