Descrição detalhada

Representa uma marca.

A classe Marca armazena as informações de uma marca, incluindo o nome, descrição, data de criação, país de origem e a sua relação com produtos. Esta classe permite gerir as marcas associadas a produtos no sistema.

A classe oferece funcionalidades para verificar se a marca pode ser eliminada, com base na sua associação com produtos vendidos, e permite a definição de uma data de criação automática.

bool Marca.PodeSerEliminada ( List< Produto >  produtos)

Verifica se a marca pode ser eliminada.

Este método verifica se a marca pode ser eliminada com base na sua associação a produtos. A marca só pode ser eliminada se não estiver associada a nenhum produto.

Parâmetros
produtosLista de produtos para verificar se algum está associado a esta marca.
Retorna
Retorna true se a marca pode ser eliminada, ou false caso contrário.
