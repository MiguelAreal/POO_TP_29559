Carrega a lista de países a partir de um ficheiro JSON. Lê um ficheiro JSON contendo uma lista de países e retorna essa lista como uma lista de strings. Caso o ficheiro não exista ou ocorra um erro ao lê-lo, retorna uma lista vazia e exibe uma mensagem de erro.
